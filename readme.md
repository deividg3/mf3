# Travel Explorer PWA

Aplicación web progresiva (PWA) desarrollada con:

- HTML5
- CSS3
- JavaScript ES6

## Características

✔ Diseño responsive

✔ Etiquetas semánticas

✔ Variables CSS

✔ Modo oscuro

✔ Font Awesome

✔ Información dinámica generada mediante JavaScript

✔ Datos almacenados en arrays

✔ API Open-Meteo para consultar el tiempo

✔ PWA instalable

✔ Manifest

✔ Service Worker

✔ Caché offline

✔ Enlaces a Wikipedia

## API utilizada

Open Meteo:

https://open-meteo.com/

## Instalación

1. Descargar el proyecto.

2. Abrir con un servidor local.

Ejemplo:

```bash
npx serve
```

o

```bash
python -m http.server
```

3. Abrir en navegador.

4. Instalar la aplicación desde el navegador.
